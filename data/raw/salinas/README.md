# Salinas Hyperspectral Dataset

## Purpose

This public hyperspectral dataset is used to validate that the paper
reimplementation pipeline operates correctly on genuine hyperspectral data.

It is not the original beverage-stain dataset used in the target paper.

## Source

Zenodo record: 10.5281/zenodo.15771735

Original imagery source:
NASA/JPL Airborne Visible/Infrared Imaging Spectrometer (AVIRIS).
